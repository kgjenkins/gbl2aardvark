gbl2aardvark: convert GeoBlacklight 1.0 json files to Aardvark

Select your own local geoblacklight.json file to convert, or try using the GeoBlacklight fixtures (TEST) or Cornell's records (CUGIR).

You will most certainly want to review and edit the generated records! Specifically, look for the following:

  • If any output fields start with "EDIT ME", that means that the input could not be unambiguously converted to Aardvark.
  • If your input records used the old dct_isPartOf_sm field (containing a text string of a collection name), there is an option to automatically generate collection records in Aardvark, as well as links to them from the child records. The new collection records will appear at the end of the output JSON, and tend to aggregate more info than you will really want to include in the collection record, so be sure to edit! See the README for more details.

Please report any issues here: https://github.com/kgjenkins/gbl2aardvark/issues
